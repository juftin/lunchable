class lunchable.LunchMoney(access_token=None)[source]

Bases: lunchable.models.assets.AssetsClient, lunchable.models.budgets.BudgetsClient, lunchable.models.categories.CategoriesClient, lunchable.models.crypto.CryptoClient, lunchable.models.plaid_accounts.PlaidAccountsClient, lunchable.models.recurring_expenses.RecurringExpensesClient, lunchable.models.tags.TagsClient, lunchable.models.transactions.TransactionsClient

Lunch Money Python Client.

This class facilitates with connections to the Lunch Money Developer API. Authenticate with an Access Token. If an access token isn’t provided one will attempt to be inherited from a LUNCHMONEY_ACCESS_TOKEN environment variable.

Examples

from lunchable import LunchMoney

lunch = LunchMoney(access_token="xxxxxxx")
transactions = lunch.get_transactions()
Parameters

access_token (optional) – [Default]: None

Three-letter lowercase currency code in ISO 4217 format. The code sent must exist in our database. Defaults to user account’s primary currency.
